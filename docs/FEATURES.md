# 実装済み機能一覧

## コア機能
- **ランディングページ** — サービス紹介
- **Google / LINE ログイン** — ソーシャル認証
- **旅行一覧・作成** — 1旅行 = 1つの縦長ページ、検索・フィルター・ソート対応
- **テーマ選択（4種）** — 「しずか」/「写真映え」/「レトロ」/「ナチュラル」+ ダークモード
- **即時テーマプレビュー** — 編集画面でテーマ変更が即座に反映、カラースウォッチ付き
- **AI旅程生成** — 目的地・日程・スタイル・予算を指定して自動生成（Llama 3.3 70B）、参考画像対応（LLaVA）
- **カバー画像アップロード** — R2保存、OGP連動
- **招待リンク** — トークンベース、無効化・再発行可能
- **OGP動的プレビュー** — テーマ連動カード、日数バッジ・エリア表示・アクセントライン付き

## 旅程編集
- **ドラッグ&ドロップ並び替え** — アイテム・日程の順序変更
- **ビジュアルタイムライン** — 縦線 + ドットでアイテム接続、終了時刻表示
- **スクロールアニメーション** — 日程セクションのフェードイン表示
- **日程メモ** — 日ごとのメモ欄（Markdown対応）
- **思い出写真** — アイテムごとの複数枚アップロード、枚数に応じた自動レイアウト
- **地図埋め込み** — Google Maps連携、複数地点表示
- **天気予報** — 直近14日間の日程に天気アイコン・気温を表示
- **チェックイン** — 旅行中のスポットチェックイン機能
- **カレンダーエクスポート** — .ics形式でダウンロード
- **JSON/CSVエクスポート** — データバックアップ
- **変更履歴・復元** — 編集履歴の記録、スナップショット復元

## 共同編集・共有
- **共同編集者招待** — オーナー/編集者/閲覧者の権限管理
- **共有ページ** — ログイン不要で閲覧可能
- **LINE/X共有ボタン** — SNS共有
- **QRコード** — 共有リンクのQR表示

## 予算・経費
- **予算設定** — 旅行全体の予算上限、進捗バー
- **経費記録** — アイテムごとの費用入力
- **経費分割** — メンバー間の割り勘計算（均等・カスタム対応）
- **精算サマリー** — 誰が誰にいくら払うか表示

## 写真・アルバム
- **写真アルバムページ** — 旅程の写真を一覧表示
- **ライトボックス** — 写真の拡大表示、ナビゲーション
- **自動レイアウト** — 1枚=フル幅、2枚=横並び、3枚=2+1配置

## 通知・リマインダー
- **リマインダー設定** — 出発前通知（Push Notification）
- **カウントダウン** — 出発日までの日数表示

## その他
- **プロフィールページ** — アカウント情報、統計表示
- **統計ページ** — 旅行数、費用集計、テーマ別統計、訪問地ランキング
- **アーカイブ機能** — 過去の旅行を非表示に
- **複製機能** — 既存の旅程をコピー
- **テンプレート** — 公開テンプレートから新規作成
- **持ち物リスト** — カテゴリ別チェックリスト、進捗バー
- **カラーラベル** — 旅行のカテゴリ色分け
- **音声入力** — アイテムタイトルの音声入力
- **印刷対応** — 印刷用スタイル（アニメーション・UI非表示）
- **PWA対応** — オフライン表示、ホーム画面追加
- **フィードバック** — 星評価・コメント
- **旅程枠購入** — Stripe連携の追加枠購入
